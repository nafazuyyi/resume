Mobile App Project

  • Menu video berisi video animasi pembelajaran
  • Menu kuis berisikan kuis mengenai materi yang telah dirangkum dalam poster rangkuman maupun video materi
Aplikasi dibangun dalam bahasa REACT NATIVE.

information System Project

Sistem informasi manajemen sekolah berbasis website untuk Pondok Pesantren Hubbul Khoir Sukoharjo dibangun menggunakan framework Laravel 10, sistem ini dibuat agar proses administrasi pesantren Hubbul Khoir dapat terintegrasi dengan baik sehingga proses pelayanan dan manajemen sekolah akan berjalan dengan lebih efektif. Aplikasi ini merupakan software dengan cakupan fitur

  • Web memiliki multi role, dimana setiap role memiliki akses terhadap fitur yang berbeda-beda. Role tersebut antara lain role siswa, guru, akuntan, admin dan super admin
  • Web memiliki fitur jadwal akademik, administrasi tagihan, fitur input siswa, kenaikan kelas dan kelulusan siswa, input users, fitur kelas, fitur input mata pelajaran, fitur untuk mengelola ujian.
  • dan fitur-fitur lainnya
Sistem informasi ini dibangun dengan bantuan framework Laravel dimana framework ini menggunakan bahasa php, html, css, js. Koneksi database menggunakan bahasa SQL dengan RDBMS MySQL
Link Website Sekolah: https://portal.hubbulkhoir.sch.id/login
